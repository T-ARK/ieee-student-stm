## Project Purpose
This project was created as a short academic assignment to design a real-world organizational website for an IEEE Student Branch.  
The focus was on responsive design, proper structure, and professional presentation.

## How to Run Locally
1. Clone or download the repository
2. Open the project folder
3. Open `index.html` in any web browser

No installation or dependencies are required.
